Getting Started

  1. Download: Clone the repository or download the latest version from the GitHub page.
  2. Installation: Ensure that .NET and WinUI 3 dependencies are installed, then run the application.
  3. Run as Administrator: For full functionality, right-click and select "Run as Administrator".
  4. Explore: Navigate through the menu to access optimization, cleanup, and customization options.

About the Project

PWin11 Tweaker is a lightweight open-source utility that allows you to customize various aspects of Windows 11 without delving into complex system settings or third-party software. Focused on performance optimization, system cleanup, privacy protection, and interface customization, this tool simplifies the process of tailoring your operating system to your specific needs. Intuitive navigation and responsive design make it accessible to all, while advanced features cater to tech enthusiasts.

Key Features

  • System Optimization: Boost your system's performance by tweaking settings like startup programs, background processes, and visual effects.
  • Privacy Protection: Control your data by disabling telemetry, clearing activity history, and managing location services.
  • Interface Customization: Personalize the look of Windows 11 by switching themes, customizing the taskbar, and more.
  • Temp Cleaner: Safely remove temporary files, recycle bin contents, browser cache, and Windows update leftovers to free up disk space.
  • File Explorer Enhancements: Customize File Explorer behavior, enable hidden features, and improve navigation efficiency.
  • Performance Tuning: Optimize gaming performance, power plans, and memory usage with a few clicks.
  • User-Friendly Navigation: Seamlessly switch between settings pages using a modern NavigationView interface with smooth, detailed animations.
  • Admin Alert: Notifies you if the application is launched without administrator privileges, ensuring full access to system settings.

System Requirements

  • Operating System: Windows 11 (64-bit version)
  • Platform: .NET 9.0
  • Permissions: Administrator privileges recommended
